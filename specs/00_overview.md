# 概要 (Overview)

## コンセプト
devrosは、ROS 2向けの高速ビルド・デプロイツールです。`colcon`を置き換え、以下のコア機能を提供することを目的としています。

1.  **高速なインクリメンタルビルド**: スマートなキャッシュとRustのパフォーマンスを活用します。
2.  **実体化 (Materialization)**: symlink-installされたディレクトリを実ファイルに変換し、ポータブルなデプロイを可能にします。
3.  **プロセス管理**: ROS 2 launchファイルやプロセスを管理するための組み込みスーパーバイザーを提供します。
4.  **自動リロード**: 継続的な開発フィードバックループのためのウォッチモードを提供します。

## 目標
- **パフォーマンス**: 環境セットアップとビルド実行のオーバーヘッドを最小限に抑えます。
- **ポータビリティ**: 複雑なセットアップなしに他のマシン（ロボットなど）へデプロイ可能な、スタンドアロンな `install` ディレクトリを生成します。
- **ユーザビリティ**: 開発者体験向上のため、モダンなCLIとTUIを提供します。
